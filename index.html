Course Catalog Description

This course will cover the creating of Web pages and sites using HTMLS, css, Javascript and graphical application. Networked multimedia distribution technologies are also exlpred. The design of effective Web site includig page layout, user interface design, graphic design, content flow and site structure as well as management and design consideration are addressed. Students design and create major Web Site with multiple pages and cross-linked structures. Prerequisites: [none] Credit: 2-2-3 Semester Hours

Course Outcomes:

Students completing this course will be able to:

  • Recognize HTML, CSS and Javacript mark-up and code in a web page/application
  • Select the proper mark-up tags or code to achieve a particular result.
  • Identify impoperly used mark-up code.
  • Produce modern standards complient web pages.
  • Deploy web pages to a public server.
